2. The __init__ Method - The Constructor 
What does it do?
This special method runs automatically when you create a new account. It sets up the initial values.
pythondef __init__(self, account_holder, account_number, initial_balance=0):
    self.account_holder = account_holder
    self.account_number = account_number
    self.balance = initial_balance
How it works:

self refers to the specific account being created
When you write: account1 = BankAccount("Alice", "ACC001", 1000)

self.account_holder becomes "Alice"
self.account_number becomes "ACC001"
self.balance becomes 1000




3. Instance Variables - Account Data 
pythonself.balance = initial_balance
self.transaction_history = []
What are these?

Each account object has its own data
self.balance stores how much money is in THIS specific account
self.transaction_history is a list that records all transactions

Example:

Alice's account has balance = 1000
Bob's account has balance = 500
They're separate and don't affect each other!


4. Methods - Actions You Can Perform 
a) Deposit Method
pythondef deposit(self, amount):
    if amount <= 0:
        print("Error: Amount must be positive!")
        return False
    
    self.balance += amount
    self.transaction_history.append(f"Deposit: +${amount:.2f}")
    return True
Step-by-step flow:

Check if amount is valid (must be positive)
Add money to balance: self.balance += amount
Record the transaction in history
Return True to show success

Example:
pythonaccount1.deposit(500)
# balance was 1000, now becomes 1500

b) Withdraw Method
pythondef withdraw(self, amount):
    if amount > self.balance:
        print("Insufficient funds!")
        return False
    
    self.balance -= amount
    return True
Key features:

Validation: Checks if you have enough money
Deduction: Subtracts from balance
Error handling: Returns False if insufficient funds


c) Transfer Method
pythondef transfer(self, recipient_account, amount):
    self.balance -= amount  # Take from sender
    recipient_account.balance += amount  # Give to recipient
How it works:

Takes another BankAccount object as parameter
Removes money from sender (self)
Adds money to recipient
Records transaction in both accounts
