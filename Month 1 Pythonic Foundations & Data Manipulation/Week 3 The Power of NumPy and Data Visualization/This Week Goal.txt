Week 3: The Power of NumPy and Data Visualization

Day 1: NumPy Fundamentals & Performance
Focus: Array basics, vectorization, and understanding NumPy's speed advantage
Learning Goals:

Create and manipulate NumPy arrays
Understand vectorization vs loops
Basic array operations and indexing

Projects:

Performance Comparison - Compare loop vs vectorized operations
Histogram of Normal Distribution - Generate random data and visualize

Practice Tasks:

Create arrays using np.array(), np.arange(), np.linspace()
Practice array slicing: arr[start:stop:step]
Calculate mean, median, std using NumPy functions


Day 2: Array Operations & Image Manipulation
Focus: Advanced indexing, array reshaping, and practical applications
Learning Goals:

2D array manipulation
Boolean indexing and fancy indexing
Working with image data as arrays

Projects:

Image Manipulator - Load, manipulate, and transform images

Practice Tasks:

Reshape arrays with .reshape() and .transpose()

Stack arrays with np.vstack(), np.hstack()
