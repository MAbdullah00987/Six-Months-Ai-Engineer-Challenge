Multi-Panel Dashboard: Create a single figure with multiple subplots (e.g., a line chart, a bar chart, and a scatter plot) using Matplotlib.

Step-by-Step Implementation Guide
Step 1: Import Required Libraries
import matplotlib.pyplot as plt
import numpy as np
Step 2: Prepare Your Data
Create sample datasets for each visualization type.

# Line chart data
months = ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun']
sales = [45, 52, 48, 65, 70, 78]
Step 3: Create Figure with Subplots
Use plt.subplots() to create a grid layout (2 rows, 2 columns).

fig, axes = plt.subplots(2, 2, figsize=(14, 10))
fig.suptitle('Sales Analytics Dashboard', fontsize=16)
Step 4: Create Individual Charts
Plot each chart type in its designated subplot position.

# Line chart: axes[0, 0]
# Bar chart: axes[0, 1]
# Scatter plot: axes[1, 0]
# Fourth panel can be hidden or used for another chart
Step 5: Customize and Display
Add styling, adjust spacing, and show the final dashboard.

plt.tight_layout()
plt.show()
